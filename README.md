# Workshop ReactJS Hour

## Criar projeto:
```
npx create-react-app workshop-reactjs-hour
```

## Clonar repositório:
```
git clone https://github.com/Markimg22/workshop-reactjs.git
```

## Mudar branch:
```
git checkout config-project
```

## Instalar dependências:
```
npm install
```
```
npm i
```

## Iniciar aplicação:
```
npm start
```
